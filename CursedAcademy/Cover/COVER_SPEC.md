# Cover Template Specs for KDP

## Trim Size: 6" x 9" (inches)
## PDF Cover Size: 12.875" x 9" (with spine)

### Dimensions Breakdown:
- **Page count estimate:** ~280 pages
- **Spine width:** ~0.875" (87.5 points)
- **Total cover width:** 6" + 0.875" + 6" = 12.875"
- **Total cover height:** 9"

### At 300 DPI (print resolution):
- **Width:** 3863 pixels
- **Height:** 2700 pixels
- **Spine:** 262 pixels

---

## Cover Design Elements

### Front Cover:
1. **Background:** Dark gradient (deep navy/black with subtle mystical pattern)
2. **Title:** "THE BINDING" - Large gold serif font (Cinzel or similar)
3. **Subtitle:** "A Dark Fantasy Romance" - Smaller, lighter gold
4. **Series:** "The Cursed Academy, Book 1" - Copper/bronze color
5. **Tagline:** "Some bonds are meant to break. And others are meant to last." - Italic
6. **Author:** "Karma Bailey" - Gold, bottom center
7. **Genre tags:** "ENEMIES TO LOVERS • FATED MATES" - Small, bottom
8. **Emblem:** Stylized phoenix outline in faint gold (upper center)

### Spine:
- Title vertically (rotated 90°)
- Author name vertically

### Back Cover (for full wrap):
- Blurb/synopsis
- Author bio
- QR code for "Look Inside"
- Amazon logo

## How to Use:

### Option 1: Canva (Recommended)
1. Go to canva.com → Create Custom Size
2. Enter: 3863 x 2700 pixels (or 12.875" x 9" at 300 DPI)
3. Use the spec above to design
4. Download as PDF (PDF/X-4) for KDP

### Option 2: KDP Cover Creator
1. Go to KDP → Add Paperback
2. Use their cover creator
3. Or upload a template from Canva

### Option 3: HTML to Image
1. Open `cover-template.html` in a browser
2. Take a screenshot at full resolution
3. Or use browser print-to-PDF
